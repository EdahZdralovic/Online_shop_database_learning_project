** Uputstvo za  pokretanje Web aplikacije:

1.Otvorite folder Zadaca3Web 
2. Kopirajte folder Zadaca3Edah (nalazi se unutar foldera Zadaca3Web)
 u folder htdocs (ako koristite XAMPP).
3.Pokrenite XAMPP Control Panel i startujte servise:
Apache (za pokretanje PHP koda).
MySQL (za bazu podataka)

4. Importujte bazu:
Otvorite phpMyAdmin u browseru (http://localhost/phpmyadmin).
Importujte sql fajl online_trgovina (nalazi se u web program/Baza_import).
Baza nema šifru port je standardan.

Otvorite aplikaciju u browseru:
URL: http://localhost/online_trgovina_z4/index.php
